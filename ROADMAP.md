# Roadmap

Living roadmap. The authoritative, continuously-updated version is [Issue #2](https://github.com/djdarcy/infinite-glyphs/issues/2) (evergreen).

| Phase | Focus | Status |
|-------|-------|--------|
| 1 | Cursive script engine + 3-regime architecture | ✅ done (v0.1.0) |
| 1 | SVG/PNG rendering + TTF baking + CLI | ✅ done (v0.1.0) |
| 2 | Polish cursive set (cap notches, figure-8 fill, weight/spacing) | ⏳ next |
| 2 | Prime-Square-Sum integration (render `stf(n)` triangles in base `tri(n)`) | ⏳ planned |
| 3 | Cellular-automata regime wired into the provider (alternative script) | 🔮 planned |
| 3 | Font sets for bases > 65,535 glyphs (multi-file) | 🔮 planned |
| 4 | OTF/CFF output, kerning, ligature-style connections | 🔮 future |

See [CHANGELOG.md](CHANGELOG.md) for shipped changes.
